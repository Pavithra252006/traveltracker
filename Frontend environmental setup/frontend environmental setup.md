# Frontend Environment Setup

## Project Title

Travel Tracker: A Mobile App for Capturing Trip Information

## Objective

To set up the frontend development environment required for developing the Travel Tracker mobile application.

---

## Frontend Technology

* Programming Language: Java
* IDE: Visual Studio Code (VS Code)
* Platform: Android Application Development

---

## Software Requirements

1. Java Development Kit (JDK)
2. Visual Studio Code (VS Code)
3. Extension Pack for Java

---

## Installation Steps

### Step 1: Install Java JDK

Download and install Java Development Kit (JDK).

Verify installation:

```bash
java -version
javac -version
```

### Step 2: Install Visual Studio Code

Download and install Visual Studio Code.

### Step 3: Install Java Extension

Open VS Code and install:

* Extension Pack for Java

### Step 4: Create Project Folder

```text
TravelTracker
│
└── frontend
```

### Step 5: Create Java File

File Name:

```text
MainActivity.java
```

Sample Code:

```java
package frontend;

public class MainActivity {

    public static void main(String[] args) {
        System.out.println("Travel Tracker Application");
    }
}
```

### Step 6: Execute the Program

Run the Java file in VS Code.

Expected Output:

```text
Travel Tracker Application
```

---

## Environment Verification

The frontend environment setup is verified when:

* Java is installed successfully.
* VS Code is configured with Java extensions.
* The sample Java application executes successfully.
* Output is displayed in the terminal.

---

## Expected Outcome

A fully configured frontend development environment capable of developing the Travel Tracker application's user interface and frontend modules, including Login, Dashboard, Trip Tracking, and Travel History screens.

---

## Conclusion

The frontend environment for the Travel Tracker project has been successfully configured using Java and Visual Studio Code. The setup is ready for developing and testing frontend application components.
